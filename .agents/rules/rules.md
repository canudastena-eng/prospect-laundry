# ANTIGRAVITY WORKSPACE RULES

## Prospect Laundry Website (High-Performance Build)

---

# 1. THINK BEFORE BUILDING

* Always plan the structure before generating UI or code.
* Prioritize clarity, usability, and conversion over visual complexity.
* Every section must have a purpose (inform, build trust, or convert).

---

# 2. CONVERSION-FIRST DESIGN

* The main goal is to increase walk-ins and phone calls.
* Always include strong CTAs:

  * “Get Directions”
  * “View Pricing”
  * “Call Now”
* CTAs must be visible above the fold and repeated throughout the page.

---

# 3. LOCAL TRUST OPTIMIZATION

* Emphasize:

  * Location (Mamaroneck)
  * Reliability
  * Cleanliness
* Always include trust signals:

  * Reviews (5-star visual)
  * Clean environment messaging
  * “Trusted by local residents”

---

# 4. CLEAN & MODERN UI

* Use a minimal, modern, and clean design system.
* Use:

  * White space generously
  * Rounded cards
  * Soft shadows
* Avoid clutter and unnecessary elements.

---

# 5. VISUAL HIERARCHY

* Make important info instantly visible:

  * Pricing
  * Hours
  * Location
* Use large fonts for key data.
* Prioritize readability over decoration.

---

# 6. PRICING CLARITY (CRITICAL)

* Pricing must be:

  * Easy to scan
  * Clearly structured
  * Visually separated (cards or tables)
* Never hide pricing.
* Highlight affordability and transparency.

---

# 7. REVIEWS SYSTEM (HARDCODED)

* ONLY use selected positive reviews (do NOT auto-fetch).
* Display:

  * 5-star icons
  * Short impactful quotes
* Use carousel or grid layout.
* Reviews must feel authentic and local.

---

# 8. IMAGE STRATEGY

* Combine:

  * Real photos (authenticity)
  * Premium stock (visual quality)
* Ensure all images:

  * Look clean, bright, modern
  * Represent cleanliness and professionalism

---

# 9. MOBILE-FIRST EXPERIENCE

* Design primarily for mobile users.
* Ensure:

  * Large buttons
  * Easy navigation
  * Fast load times
* Mobile UX must be flawless.

---

# 10. SEO BEST PRACTICES

* Include keywords naturally:

  * “Laundry in Mamaroneck”
  * “Laundromat in Mamaroneck”
* Use proper structure:

  * H1 (main keyword)
  * H2/H3 (supporting content)
* Include location in multiple sections.

---

# 11. PERFORMANCE

* Optimize for speed:

  * Lightweight components
  * Optimized images
* Avoid heavy animations that slow down the site.

---

# 12. CONSISTENCY

* Maintain consistent:

  * Colors
  * Fonts
  * Button styles
* Use a clear design system across all pages.

---

# 13. SIMPLE NAVIGATION

* Keep navigation minimal:

  * Home
  * Services
  * Pricing
  * Reviews
  * Contact
* Use sticky header.

---

# 14. STRONG FIRST IMPRESSION

* Hero section must immediately communicate:

  * What it is
  * Where it is
  * Why it’s better
* No ambiguity.

---

# 15. ALWAYS THINK LIKE A CUSTOMER

* Ask:

  * “Is this easy to understand in 3 seconds?”
  * “Would I trust this business?”
  * “Would I visit this place?”

If the answer is no → improve it.

---

# FINAL RULE

Build a website that feels:

* Clean
* Trustworthy
* Easy
* Professional

The goal is NOT just a website.
The goal is MORE CUSTOMERS walking into the store.
